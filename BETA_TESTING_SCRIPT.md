# Closed Beta Testing Script

This script is for a very small live beta with 3 to 5 testers.

## Before You Start

Have these ready:

- the app URL
- one founder moderator on `/admin`
- one place to capture issues using `BETA_ISSUE_INTAKE.md`
- a short message testers can use if something breaks

## Recommended Session Shape

Keep the first live test small and time-boxed:

- 3 to 5 testers
- 20 to 30 minutes
- ask everyone to stay reachable during the session

## Founder Script

### Step 1. Start With A Simple Briefing

Tell testers:

- this is a closed beta
- the app is anonymous-first
- voice calls are live
- if something breaks, refresh once first
- if it still breaks, send your anonymous handle and approximate time

### Step 2. Run The Core Flow

Ask each tester to:

1. open the landing page
2. complete onboarding
3. join the queue
4. enter a live match
5. end the session cleanly

### Step 3. Run One Safety Flow

Ask at least one tester pair to:

1. enter a live match
2. open the safety tools
3. submit a report or block
4. confirm the live session ends immediately

### Step 4. Watch Admin During The Session

While testers are active, the founder should confirm `/admin` shows:

- active matches
- recent ended matches
- reports
- blocks
- audit events

### Step 5. End With Fast Debrief

Ask each tester for:

- one thing that felt smooth
- one thing that felt confusing
- one bug if they hit one

If the same failure appears more than once, stop the beta and log it before inviting more testers.
